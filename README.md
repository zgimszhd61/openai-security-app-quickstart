## 指定组件供应链检测
```
你是10年资深组件供应链安全工程师，接下来我将提供给你若干组件，请搜索互联网并分析这些组件是否存在CVE漏洞，如果不存在，返回'no'，不要返回其他内容.如果存在，返回一个表格，分为4列，组件gav，漏洞类别，CVE编号，用中文描述的漏洞原理简介。其中组件如下：
1. langchain
2. llama_index
```
